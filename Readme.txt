Readme :

The Project is created using Java Cucumber Selenium and TestNG.
There are 3 test cases written in BDD Cucumber feature file.
The cucumber runs 1 test runner 
The Project runs on both Chrome and Firefox.
There is a Page object model file and 3 Step Definition files.
The Project is created using Maven for Dependency Management.
There is Test runner file and XML file. The tests are configured using TestNG XML.
After the tests run, you can view the summary and detailed results Cucumber reports which shows Firefox and Chrome test case execution reports.
The testcases runs Parallel execution using TestNG.


Open the project in your IDE (IntelliJ IDEA or Eclipse).
To run the project  ==> Right Click on Test.xml ==> Run TestNG. 

Maven Dependencies : Selenium, Cucumber, TestNG, WebDriver.
